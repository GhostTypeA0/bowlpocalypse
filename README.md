# bowlpocalypse
This is a bowling application being designed to track my very specific interests in consistency and skills


BOWLPOCALYPSE,
is an application I am building as I descend into my bowling journey.
I am here to experience HELL!!!

My true goals are to bowl 100 perfectly formed hooks in a row, and my streak RESETS if I mess up at all!!!
Despite this, it's really nice to record the other aspects as well such as strikes, strike streaks, etc
because it's something else that i'm really curious about!!!

For the time being this app is meant to be simple, I plan to make it a downloadable app off the sight,
however I also intend on hooking it up to AWS, and JavaWebServers, DNS, and a postgres database.

I will make user account creation for any other users who wish to use my BOWLING HELL application.

In the future, I will likely add some other light features to the application
because I want to record my game scores and averages and such. But that won't be for a while.

On top of that, this app is going to stay simple and easily usable. That is my overall goal.
